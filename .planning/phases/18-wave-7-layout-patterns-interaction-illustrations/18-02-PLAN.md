    Create `src/AppBar.tsx`:

    ```typescript
    export type AppBarVariant = "minimal" | "withSearch" | "default" | "centered";

    export interface AppBarProps {
      variant?: AppBarVariant;           // default "default"
      scrolled?: boolean;                // default false — consumer passes via scroll listener
      logo?: ReactNode;                  // custom logo; renders ink box + "DS" text if omitted
      nav?: ReactNode;                   // nav links (default + centered variants)
      actions?: ReactNode;               // right-side actions (avatar, notifications)
      onSearchChange?: (value: string) => void;  // withSearch variant callback
      searchPlaceholder?: string;        // default "Search..."
      className?: string;
      style?: CSSProperties;
    }
    ```

    Implementation notes:
    - Root element: `<header className={`ds-atom-appbar ${className ?? ''}`} data-variant={variant} data-scrolled={scrolled} style={...}>`
    - Inline styles handle the scrolled transition (background, backdropFilter, boxShadow, borderBottom)
    - scrolled=false: background `var(--surf-2)`, borderBottom `1px solid transparent`, boxShadow `none`
    - scrolled=true: background `rgba(255,255,255,.92)`, backdropFilter `blur(14px)`, borderBottom `1px solid var(--rule)`, boxShadow `0 4px 16px rgba(0,0,0,.04)`
    - All transition: `all 0.2s ease`
    - `withSearch` variant: renders `<input type="search" placeholder={searchPlaceholder ?? 'Search...'} onChange={e => onSearchChange?.(e.target.value)} className="ds-atom-appbar-search" />`
    - Default logo: `<div className="ds-atom-appbar-logo">` with ink box + "DS" in amber monofont
    - `forwardRef<HTMLElement, AppBarProps>`
